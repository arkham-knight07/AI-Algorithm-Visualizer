📚 Learn Mode

Understand how different pathfinding algorithms work.

BFS (Breadth-First Search)

Approach: Explores level by level, checking all neighbors before moving deeper.

Pros: Guaranteed shortest path, systematic exploration

Cons: Uses more memory, slower than informed algorithms

Best For: Maze solving, unweighted graphs

DFS (Depth-First Search)

Approach: Dives deep into one path, backtracks when stuck.

Pros: Uses less memory, fast exploration

Cons: May not find shortest path, can get stuck in dead ends

Best For: Maze generation, cycle detection

A* (A-Star)

Approach: Uses heuristics to guide search toward goal efficiently.

Pros: Faster than BFS/DFS, finds optimal path with good heuristics

Cons: More complex, requires good heuristic

Best For: Game pathfinding, GPS navigation
